# Imagenes del Proyecto

Esta carpeta contiene imagenes y capturas de pantalla para la documentacion del proyecto Customer Opinions ETL.

## Contenido

Aqui puedes almacenar:
- Capturas de pantalla de la ejecucion del Worker Service
- Diagramas de arquitectura
- Esquemas de base de datos
- Graficos de rendimiento
- Evidencias de pruebas

## Uso

Las imagenes almacenadas aqui pueden ser referenciadas en el README.md u otros documentos usando rutas relativas:

```markdown
![Descripcion](./images/nombre-archivo.png)
```

## Nota

Las imagenes seran incluidas en el repositorio para documentacion del proyecto.
